Setup Local Server
Diperlukan untuk akses API + auto-save ke folder
Browser membatasi file:// protocol: API call gagal CORS, dan File System Access API hanya jalan di secure context. Solusi: serve file ini lewat HTTP lokal.
RECOMMENDED
VS Code · Live Server Extension
  1. Install extension "Live Server" by Ritwick Dey
  2. Buka folder berisi kartu-studio.html di VS Code
  3. Right-click file → "Open with Live Server"
  4. Browser otomatis terbuka di http://127.0.0.1:5500
🐍 Python (Mac / Linux)
cd ~/Downloads
python3 -m http.server 8000
Buka http://localhost:8000/kartu-studio.html
📦 Node.js (semua OS)
cd [folder]
npx serve
URL muncul di terminal, biasanya http://localhost:3000
🐘 PHP (sudah ada di Mac)
cd [folder]
php -S localhost:8000
Catatan: Server backend Laravel (absensi.ozodemo.my.id) tetap perlu mengizinkan CORS dari origin localhost. Kalau masih kena CORS, tambah Access-Control-Allow-Origin: * di response API, atau gunakan domain yang sama.
